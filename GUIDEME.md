# Guide — Smart Doorbell

## Prerequisites

- Python 3.10+ (for Python projects) / Node.js 18+ (for Next.js projects)
- Git

## Setup

```bash
# Clone the repository
git clone https://github.com/keerthishree20/smart-doorbell.git
cd smart-doorbell
```

## Running the Project

```bash
pip install -r requirements.txt
python main.py
```
